# Data Set Descriptions and Links
- City Courts and Rinks Data Set Link: https://data.wprdc.org/dataset/city-of-pittsburgh-courts/resource/a5b71bfa-840c-4c86-8f43-07a9ae854227
  - Used to determine data regarding how many courts/rinks there are in a neighborhood. The more courts/rinks, the "better"!
- City of Pittsburgh Trees Data Set Link: https://data.wprdc.org/dataset/city-trees/resource/1515a93c-73e3-4425-9b35-1cd11b2196da
  - Used to determine data regarding how many trees are in a specific neighborhood. The more trees, the "better"!
- Pittsburgh Police Arrests (UPDATED 2024-2025) Data Set Link: https://data.wprdc.org/dataset/pbp_arrest_data_2024_2025
  - Used to determine data regarding how many police arrests occur in specific neighborhoods in Pittsburgh. The less arrests, the "better" (or, alternatively: the more arrests, the "worst")!
